Welcome to QuizStream!

QuizStream is a hobby project designed to turn YouTube videos into interactive quizzes. Whether you're a teacher, content creator, or just someone who enjoys learning, QuizStream helps you create quizzes with minimal effort.

What Does QuizStream Do?

All you need is a YouTube link. QuizStream handles the rest—automatically generating questions based on the video’s content. You can customize the quiz format and create something engaging in just a few clicks.

Features

  • Automatic Quiz Generation: Quickly turn videos into quizzes.
  • Multiple Question Types: Choose from different formats like multiple choice.
  • User-Friendly Interface: Clean and easy-to-navigate design.
  • Powerful Backend: Reliable technology that handles quiz generation and data storage.
  • AI-Powered Content Analysis: Smart algorithms that generate relevant questions.

How It Works

QuizStream is built with a modern setup:

  • Frontend: An Angular interface that’s smooth and easy to use.
  • Backend: A Spring Boot service that manages all the data and handles requests.
  • AI Service: Python-based processing that turns video content into quiz questions.
  • Database: A reliable PostgreSQL database to store quizzes and user information.

Why Use QuizStream?

Whether you’re teaching, preparing content, or just want to test your knowledge, QuizStream makes quiz creation simple and fun. It’s a small hobby project aimed at making learning more interactive.

This project is open source and available on GitHub. Check it out and contribute at QuizStream on GitHub.
