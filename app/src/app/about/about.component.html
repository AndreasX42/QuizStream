Welcome to QuizStream!

QuizStream is a project designed to turn YouTube videos into interactive quizzes. Whether you're a teacher, content creator, or just someone who enjoys learning, QuizStream helps you create quizzes with minimal effort.

What Does QuizStream Do?

All you need is a YouTube video from which you want to create a quiz. QuizStream handles the rest, automatically generating questions based on the video’s content. You can also decide on the difficulty and the language of the quiz. At the current stage, we use OpenAI's ChatGPT to process the transcripts of the videos, so it is required to provide a corresponding API key. You can get one here. To get started, an API key is already provided to try the app out.

How It Works - The Tech Stack

QuizStream is built with a modern setup:

  • Frontend: An Angular interface that’s smooth and easy to use.
  • Backend: A Spring Boot service that manages all the data and handles requests.
  • AI Service: Python-based processing that turns video content into quiz questions.
  • Database: A reliable PostgreSQL database to store quizzes and user information.

Why Use QuizStream?

Whether you’re teaching, preparing content, or just want to test your knowledge, QuizStream makes quiz creation simple and fun. It’s a small project aimed at making learning more interactive.

This project is open source and available on GitHub. Check it out and contribute at QuizStream on GitHub.
